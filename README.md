# Japanese Reading Assistant

Japanese Reading Assistant 是一個日文長文本閱讀輔助工具。  
它可以將日文小說或文章轉換成適合學習者閱讀的逐字拆解格式，包含原句、詞彙、假名、中文語意與自然翻譯。

## 專案動機

我在閱讀日文小說時發現，對中級日文學習者來說，困難不只是單字，而是長句中的修飾關係、句尾語氣與上下文理解。

一般翻譯工具可以直接給出中文翻譯，但學習者很難知道一句話是如何被理解出來的。

因此我設計了一個工具流程，將日文長文本轉換成可閱讀、可複習的逐字拆解教材。

## 功能

- 從 EPUB / TXT 抽取日文文本
- 清理 ruby、furigana 與 HTML 標籤
- 將長文本切分成自然段落
- 根據 API 限制合併成適合處理的 chunk
- 使用固定 prompt 產生逐字拆解
- 輸出 Markdown 格式的學習文件
- 合併多個輸出檔案成完整教材

## 處理流程

```text
EPUB / TXT
↓
文字抽取
↓
文本清理
↓
自然段落切分
↓
chunk 合併
↓
API 批次處理
↓
Markdown 輸出
↓
最終文件合併
