<<<FILE 001 | 1-8>>>

【1】
朝の教室には、まだ誰もいなかった。

【2】
窓際の席だけが、白い光を受けていた。

【3】
彼女は鞄を机の横に置いて、小さく息を吐いた。

【4】
昨日の雨で、校庭には浅い水たまりが残っていた。

【5】
廊下の向こうから、誰かの足音が近づいてきた。

【6】
「早いね。」

【7】
振り向くと、彼が眠そうな顔で立っていた。

【8】
「そっちこそ。」

<<<END FILE 001>>>


<<<FILE 002 | 9-16>>>

【9】
彼女はそう言って、少しだけ笑った。

【10】
彼は隣の席に座り、まだ温かい缶コーヒーを机に置いた。

【11】
教室の時計は、七時四十分を指していた。

【12】
いつもなら、二人ともこの時間にはまだ家にいる。

【13】
けれど今日は、文化祭の準備で早く集まることになっていた。

【14】
黒板には、昨日書いたままの予定表が残っていた。

【15】
紙を切る係、看板を作る係、買い出しに行く係。

【16】
どれも大した仕事ではないはずなのに、なぜか少しだけ特別に見えた。

<<<END FILE 002>>>


<<<FILE 003 | 17-24>>>

【17】
「昨日、帰るの遅かったでしょ。」

【18】
彼女が聞くと、彼は缶コーヒーのふたを開けながらうなずいた。

【19】
「家に帰ってからも、少しだけデザインを直してた。」

【20】
「真面目だね。」

【21】
「そういうわけじゃないよ。」

【22】
彼は困ったように笑った。

【23】
その笑い方を見て、彼女は何も言えなくなった。

【24】
窓の外では、運動部の掛け声が少しずつ大きくなっていた。

<<<END FILE 003>>>


<<<FILE 004 | 25-32>>>

【25】
朝の学校は、昼間よりも音がよく響く。

【26】
机を動かす音も、椅子を引く音も、遠くの階段を上る音も、全部はっきり聞こえた。

【27】
彼女は予定表を見ながら、自分の名前を探した。

【28】
そこには、「展示説明」とだけ書かれていた。

【29】
「私、人前で話すの苦手なんだけど。」

【30】
「じゃあ、練習しようか。」

【31】
彼は当たり前みたいに言った。

【32】
その言葉があまりにも自然だったので、彼女はまた少し笑ってしまった。

<<<END FILE 004>>>


<<<FILE 005 | 33-40>>>

【33】
最初の練習は、想像していたよりもずっとぎこちなかった。

【34】
彼女はメモを持ったまま、同じ言葉を何度も言い直した。

【35】
彼はそのたびに、うなずいたり首をかしげたりした。

【36】
「今のは少し早口だったかも。」

【37】
「やっぱり？」

【38】
「でも、内容はちゃんと伝わってる。」

【39】
その一言で、彼女の肩から少し力が抜けた。

【40】
窓の外の空は、さっきよりも明るくなっていた。

<<<END FILE 005>>>


<<<FILE 006 | 41-48>>>

【41】
八時を過ぎるころには、教室に少しずつ人が集まり始めた。

【42】
眠そうな顔の生徒や、朝から妙に元気な生徒が入ってきた。

【43】
誰かが大きな袋を床に置き、誰かが紙の束を配り始めた。

【44】
教室の静けさは、あっという間に消えていった。

【45】
彼女はメモを折りたたみ、制服のポケットに入れた。

【46】
「あとでまた練習しよう。」

【47】
彼がそう言うと、彼女は小さくうなずいた。

【48】
文化祭の一日は、まだ始まったばかりだった。
